# HRM
### HRM (Human Resource Managemen) project

## Стэк разработки:

### Фронтенд (Next.js 14+):
1) App Router (/app) для маршрутов.
2) State management: React Query (для API кэшинга)
Query + Zustand для глобального состояния (роли, уведомления).
3) UI: Shadcn/UI

### Бэкенд (Spring Boot 3+):
БД: PostgreSQL (с JPA/Hibernate).
Аутентификация: Spring Security с JWT (для API).
